## The Problem

You know meditation would help. But sitting in silence feels impossible when your mind won't stop racing.

**Aura** meets you where you are. Guided sessions for stress, sleep, and focus. Ambient sounds when you need background calm. Sleep stories when counting sheep isn't cutting it.

---

## What You Get

| Feature | Description |
|---------|-------------|
| **Guided Meditations** | Sessions for stress, sleep, focus & more |
| **Ambient Sounds** | Nature sounds and calming music |
| **Sleep Stories** | Drift off with soothing narratives |
| **Mood Tracking** | Monitor your emotional well-being |
| **Timer** | Unguided meditation with bells |
| **Streaks** | Build consistent mindfulness habits |
